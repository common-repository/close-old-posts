== Description ==

Closes comments on old posts on the fly, without any DB queries. By default it's 14 days, change that setting by editing the plugin file.

Plugin is not needed for WordPress 2.7 and up, as the functionality is built into core.

== Installation ==

1. Upload the files to the `/wp-content/plugins/close-old-posts/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
